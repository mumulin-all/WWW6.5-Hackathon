项目名：symbreath共生呼吸
简介（痛点与背景//增加）：我们利用 Web3 的博弈机制与实时气象数据，建立一套跨物种的共生协议。
通过将森林的生存压力转化为个体的冥想交互，让环保从“遥远的公益”变成“切身的感知”，用户可以通过链上抵押代币完成冥想挑战，如果挑战成功将退回代币铸成NFT徽章，中途断签则质押金自动捐给真实环保慈善地址。
功能 - 质押 0.01 ETH，定义7/21挑战天数开启挑战 - 每日打卡，选择灾难模式触发冥想动效 - 挑战成功：全额退款 + 铸造专属 SBT 徽章 - 主动放弃：质押金自动捐给环保机构 - SBT 灵魂绑定，不可转让
技术栈 - 前端：React（Lovable） - 
合约：Solidity 0.8.20 / OpenZeppelin（ERC721 + ReentrancyGuard + Ownable）
 - 后端：Node.js + Express + ethers.js 
- 网络：Avalanche Fuji 测试网
