# RouteMint - 去中心化知识问答社区

## 项目简介

RouteMint 是一个创新的去中心化问答（Q&A）平台，旨在通过区块链技术和经济激励，连接寻求知识的提问者与拥有宝贵经验的回答者。在 RouteMint，用户可以就“学习”、“职业”和“创业”等关键人生领域提出问题，并通过质押以太币（ETH）设立奖金池，悬赏高质量的答案。

我们的使命是为人们在面临重要抉择时，提供一个直接、可信且有价值的经验分享渠道，打破信息壁垒，让知识和经验的价值得以体现和流转。

## 核心理念

在传统的信息渠道中，获取真实、深度且个性化的经验往往非常困难。我们相信，那些在特定领域拥有实际经验的人，他们的见解对于后来者具有不可估量的价值。RouteMint 的核心理念就是：

- **价值驱动分享**：通过经济激励，鼓励拥有宝贵经验的“过来人”分享他们的知识和见解。
- **质量决定回报**：回答的质量直接决定了回答者能获得的回报，确保平台内容的优质性。
- **信誉沉淀价值**：用户的每一次高质量分享都会累积为个人声誉，使其在社区中的影响力不断增长。

## 项目实现

RouteMint 构建在以太坊区块链之上，其核心逻辑由智能合约 `RouteMint.sol` 实现，前端则通过 `ethers.js` 与区块链进行交互。

### 技术栈

- **智能合约**: Solidity
- **区块链平台**: Ethereum
- **前端库**: ethers.js, HTML, CSS, JavaScript
- **内容存储**: 问题和答案的详细内容通过 IPFS (InterPlanetary File System) 进行去中心化存储，仅将内容的哈希（CID）上链，有效降低了链上存储成本并保证了数据的不可篡改性。

### 核心功能流程

1.  **提出问题 (Create Question)**
    -   **提问者** 在前端页面填写问题标题、选择分类（学习、职业、创业），并撰写详细的问题描述。
    -   为了激励高质量的回答，提问者需要设定一笔 **质押金 (Stake)** 和一个 **奖励池 (Reward Pool)**。质押金确保提问者会认真对待并完成整个问答流程，奖励池则用于奖励优秀的回答者。
    -   问题内容被上传至 IPFS，生成唯一的 CID。
    -   提问者通过钱包发起一笔交易，调用智能合约的 `createQuestion` 函数，将问题标题、分类、CID、质押金和奖励池金额一同提交上链。

2.  **提交答案 (Submit Answer)**
    -   **回答者** 可以在平台上浏览开放的问题。
    -   找到自己感兴趣且有经验回答的问题后，撰写答案并附上自己的“经验标签”（如“三年工作经验”、“创业公司创始人”等）。
    -   答案内容同样被上传至 IPFS。
    -   回答者调用 `submitAnswer` 函数，将答案的 CID 和经验标签提交上链。为保证公平，提问者不能回答自己的问题，且每位用户对同一问题只能回答一次。

3.  **评价答案 (Rate Answer)**
    -   **提问者** 在收到回答后，可以根据答案的质量进行 1 到 3 星的评级。
    -   评级会转换成不同的“权重”（1星=1，2星=2，3星=4），这个权重将决定该回答能从奖励池中分得多大比例的奖金。
    -   提问者调用 `rateAnswer` 函数完成评价。

4.  **关闭问题与分配奖励 (Close Question & Distribute Rewards)**
    -   当提问者认为已经收集到满意的答案，或者问答周期结束时，可以决定 **关闭问题**。
    -   提问者调用 `closeQuestion` 函数，智能合约会根据所有已评价答案的权重，按比例将 **奖励池** 中的 ETH 分配给对应的回答者。
    -   奖励分配完成后，提问者的 **质押金** 将被全额退还。如果奖励池有剩余（例如，没有回答或部分回答未被评价），剩余部分也会一并退还给提问者。
    -   回答者获得的奖励权重也会累积为他们的 **声誉值 (Reputation)**，成为其在社区内专业能力的证明。

### 安全性

智能合约集成了 OpenZeppelin 的 `ReentrancyGuard` 模块，以防止在分配奖励等涉及资金转移的关键操作中发生重入攻击，保障了平台的资金安全。

## 如何使用

1.  **安装钱包**：在您的浏览器中安装 MetaMask 或其他兼容的以太坊钱包。
2.  **连接网络**：切换到项目部署的以太坊网络（如 Sepolia 测试网或主网）。
3.  **开始互动**：
    -   访问 `routemint-ask.html` 来提出您的问题。
    -   访问 `routemint-explore.html` 浏览并回答他人的问题。
    -   访问 `routemint-me.html` 查看您自己的个人信息、声誉和历史记录。

通过 RouteMint，我们希望构建一个正向循环的知识经济生态，让每一个有价值的经验都能找到归宿，也让每一个寻求帮助的人都能找到自己的路径（Route）。
